# Molecularity

Минималистичная мобильная головоломка на графе молекул. Финальная цель — приложение
на Unity; сейчас игровое ядро (кор-луп) пишется как чистое .NET-решение в `dotnet/`,
чтобы отрабатывать логику без редактора и покрывать тестами.

## Запуск консольного раннера

```bash
cd dotnet
dotnet run --project Molecularity.Console
```
