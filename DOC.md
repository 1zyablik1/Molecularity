> - **«Сканер» удалён** → его роль закрывает предмет `RevealAll`.
> - Вместо одной бесплатной абилки — **инвентарь предметов** (RevealAll, PlusOneAll,
>   Freeze, ChainBreak, Undo), покупаемых за **золотые монеты** / выдаваемых за **дейли-квесты**.
> - **Ревайв** — это применение `PlusOneAll` (отдельной системы нет).
> - **Лимита ходов нет.** Победа = очистить поле; поражение = живая молекула ≤ 0.
> - Параметры: **Anchor −2 + «+1 соседям»**, **Shield 2 хода неуязвимости**, **Parasite −(число соседей)**.
> - Типы MVP: **Simple, Parasite, Shield, Anchor** (Virus/Teleport — вне скоупа).
> - **Бесконечный режим отложен.** Раскраска — основная **мета-игра** (слой Unity).

## 🎮 Концепция игры

### Elevator Pitch
Минималистичная головоломка для мобильных устройств, где игрок управляет сетью молекул с обратным отсчетом. Каждый клик — это стратегическое решение в мире, где информация скрыта, а время ограничено.

### Целевая платформа
- **Основная:** Android (Google Play Store)
- **Ориентация:** Вертикальная (Portrait)
- **Целевая аудитория:** Казуальные игроки 18-45 лет
- **Сессия:** 2-5 минут на уровень
- **Жанр:** Головоломка / Puzzle

### Уникальное торговое предложение (USP)
1. **Граф вместо сетки** — нелинейная структура уровней
2. **Система видимости** — значения скрыты, появляются по правилам
3. **Обратный отсчет** — каждая молекула "умирает" через N ходов
4. **Минималистичный неон** — узнаваемый визуальный стиль

## ⚙️ Основные механики

### Игровое поле
- **Структура:** Граф (сеть узлов и связей)
- **Узлы:** Молекулы с численным значением (таймер)
- **Связи:** Линии между молекулами, влияют на видимость и некоторые свойства

### Цикл хода (Game Loop)

```
1. Игрок кликает на молекулу
   ↓
2. Срабатывает абилка молекулы (on-click)
   ↓
3. Молекула удаляется с поля (больше не участвует)
   ↓
4. Все оставшиеся молекулы уменьшают счетчик (базовое свойство)
   ↓
5. Применяются пассивные свойства всех молекул
   ↓
6. Проверка проигрыша (есть молекула со значением ≤0?)
   ↓
7. Проверка выигрыша (все молекулы удалены?)
```

### Условия победы/поражения

**Победа:**
- Все молекулы удалены с поля

**Поражение:**
- Хотя бы одна молекула достигла значения ≤0
- **Важно:** Молекула, на которую кликнули, удаляется ДО проверки (можно "спасаться" последним кликом)

### Система видимости

**Базовое правило:**
- Значение становится молекулы видно, только если ее сосед был удален (еффект тумана войны), или она изначально отмечена как видимая

**Абилка "Сканер" (бесплатная):**
- **Использование:** 1 раз за уровень
- **Действие:** При клике на молекулу показывает:
    - Значение самой молекулы
    - Значения всех её соседей
- **Перезарядка:** Каждый уровень (не переносится)
- **UI:** Кнопка в HUD, становится серой после использования
- **Назначение:** Помощь казуальным игрокам, снижение фрустрации от неизвестности

### Свойства молекул

**Типы свойств:**

1. **Базовое (декремент):**
    - По умолчанию: -1 за ход
    - Может модифицироваться типом молекулы

2. **Пассивные:**
    - Срабатывают каждый ход автоматически
    - Примеры: щит, заражение, усиление декремента

3. **On-Click (абилки):**
    - Срабатывают при клике на молекулу
    - Примеры: урон соседям, лечение, телепорт

**Архитектура свойств:**
- **Тип = фиксированный набор свойств**
- Миксы (гибриды) не используются для простоты
- Каждый тип молекулы имеет строго определенное поведение

## 🧪 Типы молекул

### 1. Простая молекула (Simple)
**Декремент:** -1 за ход
**Абилка:** Нет
**Пассивная:** Нет
**Визуал:** Голубой неон
**Назначение:** Базовый тип, обучение механике
**Вводится:** Уровень 1

### 2. Молекула-паразит (Parasite)
**Декремент:** -N за ход (где N = количество соседей)
**Абилка:** Нет
**Пассивная:** Нет
**Визуал:** Малиновый/пурпурный неон, пульсация быстрее
**Назначение:** Заставляет думать о порядке кликов (убрать соседей сначала)
**Вводится:** Уровень 4-5

### 3. Молекула-щит (Shield)
**Декремент:** -0 в течение N ходов, потом -1
**Абилка:** Нет
**Пассивная:** Счетчик защиты (например, 2 хода неуязвимости)
**Визуал:** Синий неон, икона щита, мерцание при защите
**Назначение:** Выиграть время, защитить критичные зоны
**Вводится:** Уровень 10-12

### 4. Молекула-якорь (Anchor)
**Декремент:** -2 за ход (ускоренное умирание)
**Абилка:** Все соседи +1 при клике
**Пассивная:** Нет
**Визуал:** Темно-фиолетовый, тяжелая пульсация
**Назначение:** Рискованный выбор — помогает соседям, но сама умирает быстро
**Вводится:** Уровень 15-17

## Остальные не вводятся либо редизайн

### 5. Молекула-вирус (Virus)
**Декремент:** -1
**Абилка:** Нет
**Пассивная:** Каждый ход 1 случайный сосед становится вирусом (меняет тип)
**Визуал:** Зеленый неон, "щупальца" к соседям, анимация заражения
**Назначение:** Надо кликать БЫСТРО, пока не заразил всех
**Вводится:** Уровень 18-20

### 6. Молекула-телепорт (Teleport)
**Декремент:** -1
**Абилка:** Меняется местами со случайным соседом, оба получают -1
**Пассивная:** Нет
**Визуал:** Фиолетовый, спиральная анимация
**Назначение:** Переставить опасную молекулу подальше от критичных зон
**Вводится:** Уровень 22-25

### Приоритет типов для релиза

**Обязательно (MVP):**
1. Простая
2. Паразит
3. Щит

**Желательно (для 30+ уровней):**
4. Якорь
5. Вирус

**Опционально (для апдейтов):**
6. Телепорт

**Убрано из релиза:**
- ~~Генератор~~ (убран из-за неясной механики)
- ~~Репликатор~~ (слишком сложно для казуалов)

## 🎯 Режимы игры

### 1. Линейные уровни (основной режим)

**Описание:**
- Заранее спроектированные уровни
- Прогрессия сложности
- 20-50 уровней на релиз (рекомендуется 30)
- Каждый уровень — это отдельная головоломка с уникальным графом и расположением молекул

**Прогрессия:**

**Уровни 1-3: Tutorial**
- Только простые молекулы
- 3-5 молекул на поле
- Линейные графы (A—B—C)
- Обучение базовым механикам
- Невозможно проиграть
- Рисунок ракеты (линейно расположены молекулы)

**Уровни 4-10: Основы**
- Простые + паразиты
- 5-8 молекул
- Графы: звезды, треугольники, простые циклы
- Обучение стратегии порядка кликов
- Введение сканера (уровень 5-6)
- Другой рисунок (например, цветок) с более сложным расположением молекул

**Уровни 11-20: Средние**
- +Щит
- 8-12 молекул
- Сложные графы: циклы, развилки, многослойные
- Появляются молекулы с критичными таймерами (1-2 хода)
- Другие рисунки

**Уровни 21-30: Продвинутые**
- 10-15 молекул
- Асимметричные графы
- Комбинации разных типов молекул
- Другие рисунки

**Уровни 31-40: Эксперт (опционально)**

- 12-18 молекул
- Комплексные многослойные графы
- Уровни с одним правильным решением
- Другие рисунки

**Уровни 41-50: Мастер (опционально)**
- Все типы молекул
- 15-20 молекул
- Хаотичные графы
- Таймеры 1-2 хода, высокая плотность связей
- Другие рисунки

**Концепция:**
- Рисунок замаскирован молекулами (5-30 штук)
- Молекулы расставлены по форме рисунка (силуэт)
- Под каждой молекулой скрыт кусочек арта
- **НО:** молекулы **не имеют счетчиков** обратного отсчета

**Механика:**

1. **Главный экран раскраски:**
    - Показывает силуэт рисунка
    - Молекулы-точки на рисунке (20-30 штук)
    - Прогресс: "15/30 фрагментов открыто"

2. **Клик на молекулу:**
    - Открывается всплывающее окно (popup)
    - Внутри — уровень компании

3. **Решение мини-головоломки:**
    - **Победа:** Молекула исчезает → кусок арта открывается
    - **Поражение:** Можно попробовать снова (без пенальти)

4. **Прогресс:**
    - Можно проходить кусками (зашел, открыл 3 фрагмента, вышел)
    - Прогресс сохраняется

**Награда за прохождение:**
- Завершенный арт сохраняется в галерею
- Бонус: если прошел без пропусков → шейдер "золотое свечение" на рисунке

**Темы рисунков (предложения):**
- Природа: цветок, дерево, горы
- Животные: кошка, птица, рыба
- Космос: планета, ракета, звезда
- Абстракция: геометрические фигуры, мандалы
- Еда: фрукты, кофе

### 2. Бесконечный режим

**Концепция:** Тестово
Может быть переделана на постоянно растущую
Волновая система (выбрана как наиболее подходящая для казуалов)

**Механика:**

```
Волна 1:
  - 5 простых молекул
  - Значения: 3-5
  - Простой граф (звезда или цепочка)

Убил всех → пауза 0.5 секунды → "Волна 1 пройдена!"

Волна 2:
  - 7 молекул (5 простых + 2 паразита)
  - Значения: 4-6
  - Более сложный граф

Волна 3:
  - 10 молекул (микс типов)
  - Значения: 5-7

...и так далее
```

**Прогрессия усложнения:**
- **Каждая волна:** +2 молекулы
- **Каждая волна:** +1 к стартовым значениям молекул
- **Каждые 5 волн:** новый тип молекул добавляется в пул спавна
    - Волна 1-4: только простые
    - Волна 5-9: простые + паразиты
    - Волна 10-14: простые + паразиты + щиты
    - Волна 15+: все типы

**Система рекордов:**
- **Локальный рекорд:** сохраняется в SaveData

**Разблокировка:**
- Доступен после прохождения туториала

**UI:**
- Экран выбора режима: "Бесконечный режим"
- Экран поражения: лучший рекорд + реплей (показывается то как молекулы появляись, и как мы из убирали)

## 📊 Прогрессия и баланс

### Система разблокировок

**Линейные уровни (раскарска):**
- Есть граф из N - "молекул" врешин, каждая из которых это уровень, можно проходить в любом порялке, при прохождении уровня под ним открывается часть рисунка, после прохождения всех уровней открывается полный рисунок и идет переход на другой рисунок

**Бесконечный режим:**
- Разблокировка после туториала

### Абилки игрока

**Для релиза:**
- **Только Сканер** (бесплатная, встроенная)
    - 1 раз за уровень
    - Показывает значение молекулы + всех соседей
    - Перезаряжается каждый уровень

**Для апдейтов (опционально):**
- Заморозка: остановить счетчик 1 молекулы на 1 ход (за рекламу)
- +2 всем: добавить +2 ко всем молекулам (за рекламу)

### Обучение (Tutorial)

**Уровень 1: "Первый клик"**
- 3 простых молекулы, значения: 3, 2, 1
- Граф: A—B—C (линейная связь)
- Подсказка: "Нажми на любую молекулу"
- Цель: показать, что клик удаляет, счетчики уменьшаются
- **Невозможно проиграть**

**Уровень 2: "Видимость"**
- 5 молекул в форме звезды (1 центр + 4 края)
- Центр: значение скрыто (4 соседа)
- Края: значения видны (1 сосед каждый)
- Подсказка: "Значения видны только у молекул с одним соседом"
- Цель: научить думать о порядке кликов

**Уровень 3: "Таймеры"**
- 4 молекулы: разные значения (5, 3, 2, 1)
- Граф: ромб
- Подсказка: "Молекула со значением 1 опасна! Кликни на неё первой"
- Цель: научить приоритизировать опасные молекулы

**Уровень 4-5: Паразит**
- Первое появление паразита
- Подсказка: "Красная молекула — паразит. Уменьшается на количество соседей!"
- Граф: паразит в центре, 3 простых вокруг
- Цель: показать важность убирания соседей паразита

**Уровень 5-6: Сканер**
- Первое знакомство со сканером
- Подсказка: "Используй сканер, чтобы увидеть скрытые значения!"
- Граф: несколько молекул со скрытыми значениями
- Цель: научить пользоваться сканером

**Интерактивное обучение:**
- Подсказки появляются автоматически при первом столкновении с новым типом
- Можно отключить в настройках (для опытных игроков)
- Зажатие молекулы → всплывающая подсказка с описанием типа

### Глоссарий

**Доступ:**
- Кнопка в главном меню: "Энциклопедия молекул"
- Или: Settings → Glossary

**Содержание:**
- Список всех типов молекул
- Для каждого:
    - Визуальный пример (анимированная иконка)
    - Название
    - Описание декремента
    - Описание абилки/пассивки
    - "Впервые встречается: уровень X"

**Разблокировка:**
- **Все открыто сразу** (для обучения и справки)

## 💰 Монетизация

### Модель монетизации
**F2P (Free-to-Play) с рекламой + IAP**

### Реклама

**Межуровневая реклама (Interstitial):**
- Показ: каждые 2-3 пройденных уровня
- Тип: Полноэкранная (Interstitial)
- Длительность: 15-30 секунд
- **Можно отключить** через IAP "Убрать рекламу"

**Rewarded Video (добровольная реклама):**
- **Ревайв:** Проиграл уровень → предложение посмотреть рекламу для продолжения
    - Эффект: все молекулы +1 к значению, можно продолжить
    - Лимит: 1 ревайв за попытку

**Баннеры:**
- **НЕ использовать** (мешают на вертикальном экране)

### In-App Purchases (IAP)

**1. Убрать рекламу ($1.99)**
- Убирает межуровневую рекламу
- Rewarded Video остается (опционально для бонусов)
- **Самый важный IAP для релиза**

### Стратегия монетизации

**Для релиза (MVP):**
- Межуровневая реклама
- Rewarded Video для ревайва
- IAP: "Убрать рекламу"

## 🎨 UI/UX

### Главное меню

**Экран:**
```
┌─────────────────┐
│   MOLECULE      │  ← Лого (минималистичный шрифт)
│    PUZZLE       │
│                 │
│  [▶ ИГРАТЬ]     │  ← Большая кнопка (продолжить/новая игра)
│                 │
│                 │
│  ♾️  Бесконечный│
│  📚 Глоссарий   │
│                 │
│  ⚙️  Настройки  │  ← Иконка слева внизу
│  🔇 Звук        │  ← Иконка справа внизу
└─────────────────┘
```

**Анимация:**
- Фоновые пульсирующие молекулы (медленная анимация)
- Кнопки с легким свечением при нажатии (если tap)

### Экран выбора уровней

```
┌─────────────────┐
│ ←               │
├─────────────────┤
│                 │
│                 │
│     Рисунок     │  ← Силует рисунка
│      (ГРАФ)     │  (нажатие на молекулу открывает уровень)
│                 │
│                 │
│                 │
│                 │
└─────────────────┘
```

### Игровой экран

**Layout (вертикальная ориентация):**

```
┌─────────────────┐
│ ← [10]        ⚙️│  ← Топ-бар: назад, уровень, звезды, настройки
├─────────────────┤
│                 │
│                 │
│   ИГРОВОЕ ПОЛЕ  │  ← Граф молекул (центрирован)
│   (ГРАФ)        │
│                 │
│                 │
├─────────────────┤
│  Ходов: 5       │  ← Статистика (опционально)
│  [🔍] Сканер    │  ← Абилка сканера (кнопка)
└─────────────────┘
```

**Интерактивность:**
- Клик на молекулу → краткая анимация → удаление
- Зажатие молекулы (0.5 сек) → всплывающая подсказка с описанием типа
- Счетчики обновляются с анимацией (-1, -2 летящий текст)
- Клик на кнопку сканера → выбор молекулы → показ значений

**Индикаторы:**

## 🎨 Визуальный стиль

### Арт-направление

**Ключевые слова:**
- Минимализм
- Неоновая эстетика
- Пульсирующие формы
- Плавные анимации
- Темный фон

### Цветовая палитра

**Настройка**
По скольку вся игра стоится на базовых цветах, то сделать файл настроек (скриптбл) для каждого цвета, что бы можно было менять их в зависимости от темы (в постельные тона)

**Фон:**
- Глубокий темно-синий (#0a0e27) или черный (#000000)
- Легкий градиент (опционально)

**Молекулы (по типам):**
- Простая: Голубой/Cyan (#00ffff)
- Паразит: Пурпурный/Magenta (#ff00ff)
- Щит: Синий (#0080ff)
- Якорь: Темно-фиолетовый (#8000ff)
- Вирус: Зеленый (#00ff00)
- Телепорт: Фиолетовый (#a020f0)

**UI элементы:**
- Кнопки: Белый текст + неоновый контур (#ffffff + glow)
- Текст: Белый (#ffffff)
- Акценты: Соответствуют цветам молекул
- Сканер: Желтый/Золотой (#ffdd00)

### Молекулы (визуал)

**Форма:**
- Круги (основная форма)
- Диаметр зависит от важности (опционально: размер ~ значение)

**Эффекты:**
- **Пульсация:** Легкое изменение размера (0.95x - 1.05x) с периодом 1-2 секунды
- **Свечение (Glow):** Неоновый ореол вокруг (bloom эффект)
- **Внутренняя анимация:** Медленное вращение градиента (опционально)

**Счетчик:**
- Крупный шрифт в центре молекулы
- Цвет: белый или контрастный к фону молекулы
- Анимация обновления: "-1" летит вверх и исчезает

**Сканер эффект:**
- При использовании: волна расходится от выбранной молекулы
- Подсвеченные молекулы (сама + соседи): яркое свечение
- Значения показываются с анимацией (fade in)

### Связи между молекулами

**Визуал:**
- Пульсируюзие линии (слегка изогнутые и динамические)
- Толщина: 2-4 пикселя
- Цвет: Белый (#ffffff) с прозрачностью (alpha 0.5-0.7)

**Анимация (шейдер):**
- **Волны:** Бегущая волна по линии (как электрический импульс)
- **Цветовое смещение:** Легкий градиент от цвета одной молекулы к другой
- **Пульсация:** Синхронизированная с молекулами
- Скорость анимации: медленная (2-3 секунды на цикл)

**Технология:**
- Unity Line Renderer + кастомный шейдер

### Частицы и эффекты

**При клике на молекулу:**
- Взрыв частиц (цвет = цвет молекулы)
- 10-20 частиц, разлетаются радиально
- Fade out за 0.5 секунды

**При обновлении счетчика:**
- Текст "-1" (или "-2") вылетает вверх, исчезает
- Цвет: красный для уменьшения, зеленый для увеличения

**При победе:**
- Конфетти (легкое, не перегруженное)
- Цвета: микс из палитры молекул

**При поражении:**
- Красная вспышка (screen flash, alpha 0.3)
- Молекула-виновник мигает красным

**Сканер:**
- Волна расходится от молекулы (ring expand)
- Частицы-искры вокруг подсвеченных молекул

### UI стиль

**Шрифт:**
- Современный sans-serif (например: Roboto, Montserrat, Orbitron для sci-fi) (Важное чтонить лицензионное и бесплатное)
- Размеры:
    - Заголовки: 48-64px
    - Кнопки: 32-40px
    - Текст: 24-28px
    - Счетчики молекул: 36-48px

**Кнопки:**
- Rounded corners (border-radius: 8-12px)
- Неоновый контур (glow эффект)
- Hover/Tap эффект: увеличение свечения + легкий scale (1.05x)

**Иконки:**
- Минималистичные, линейные
- Цвет: белый с легким свечением

### Анимации (общие принципы)

**Timing:**
- Быстрые действия (клики): 0.2-0.3 секунды
- Обновление счетчиков: 0.4-0.5 секунды
- Пульсация: 1-2 секунды (loop)
- Волны на связях: 2-3 секунды (loop)
- Сканер: 0.5 секунды (волна расходится)

**Easing:**
- Ease-out для появления элементов
- Ease-in для исчезновения
- Elastic/Bounce для звезд на экране победы (в меру)

**Производительность:**
- Все анимации оптимизированы для мобильных устройств
- Избегать слишком сложных шейдеров
- Particle count не более 50 одновременно на экране

### Адаптация под разные экраны

**Aspect Ratio:**
- Основной: 9:16 (стандартный portrait)
- Поддержка: 9:18, 9:19 (современные смартфоны с вырезом)
- Поддержка: 9:21 (ultra-tall экраны)

**Safe Area:**
- Учитывать notch (вырез) на iPhone X и новее
- UI элементы не должны попадать под системные кнопки Android

**Масштабирование:**
- Граф молекул: автоматическое масштабирование под видимую область
- UI: canvas scaler с reference resolution 1080x1920

**Стиль:**

- Короткий радостный мотив (10-15 секунд)
- Поднимающая настроение мелодия

**6. Defeat Theme**
- Короткий грустный/напряженный мотив (5-10 секунд)
- Нисходящие ноты

**Громкость:**
- Музыка всегда на втором плане (ниже звуковых эффектов)
- Возможность отключить в настройках

### Звуковые эффекты (SFX)

**UI звуки:**
- **Клик на кнопку:** Легкий синтетический "click" (высокая нота)
- **Переход между экранами:** Свуш (whoosh)
- **Открытие popup:** Легкий "pop"

**Игровые звуки:**

**Клик на молекулу:**
- Основной звук: Синтетический "pop" (высота зависит от типа молекулы)
- Простая: средняя нота (C4)
- Паразит: низкая нота (G3)
- Щит: высокая нота + металлический звон
- Якорь: тяжелый "thud"
- Вирус: зловещий "hiss"
- Телепорт: "warp" звук

**Исчезновение молекулы:**
- Короткий "whoosh" + затухающее эхо
- Частицы: легкий треск (crackle)

**Обновление счетчика:**
- Тихий "tick" или "beep"

**Абилка молекулы:**
- Якорь (лечение соседей): восходящий "bloom"
- Вирус (заражение): зловещий "hiss" + эхо
- Телепорт: "warp"

**Сканер:**
- Активация: короткий электронный "ping"
- Волна расходится: звук "sonar ping"
- Показ значений: легкий "beep beep"

**Победа:**
- Серия восходящих нот (мажорная гамма)
- Звон (shimmer) + конфетти звук
- Длительность: 2-3 секунды

**Поражение:**
- Нисходящая нота (минорная)
- Тяжелый "thud" или "explosion"
- Длительность: 1-2 секунды

**Бесконечный режим:**
- **Переход на волну:** Восходящий "success" звук
- **Новый тип молекулы:** Уведомление "new type unlocked"

**Раскраска:**
- **Открытие фрагмента:** Приятный "chime"
- **Завершение рисунка:** Победная фанфара (короткая)

### Озвучка
